# TicTacToe-Pygame
A simple Tic Tac Toe game to provide an overview of the Python language and the Pygame module.

## How to Play
1. Clone the GitHub repository.
2. Install Pygame: `pip install pygame`
3. Run the game: `python main.py`

## References
